## Problem description
```
Given an unsorted array of integers, find the number of longest increasing subsequence.

Example 1:
Input: [1,3,5,4,7]
Output: 2
Explanation: The two longest increasing subsequence are [1, 3, 4, 7] and [1, 3, 5, 7].
Example 2:
Input: [2,2,2,2,2]
Output: 5
Explanation: The length of longest continuous increasing subsequence is 1, and there are 5 subsequences' length is 1, so output 5.
Note: Length of the given array will be not exceed 2000 and the answer is guaranteed to be fit in 32-bit signed int.
```

```java
class Solution {
    public int findNumberOfLIS(int[] nums) {
        if(nums.length == 0){
            return 0;
        }
        //dp saves the longest lIC at current index
        int[] dp = new int[nums.length];
        dp[0] = 1;
        //q (quantity) saves the possible ways of longest LIC at current index
        int[] q = new int[nums.length];
        
        q[0] = 1;
        //int ans saves number of LIS
        int ans = 1;
        //int maxval saves the current max
        int maxVal = 1;
        for(int i = 1; i < dp.length; ++i){
            int cur = 0;
            int l = 1;
            for(int j = 0; j < i; ++j){
                if(nums[i] > nums[j]){
                    if(dp[j] > cur){
                        cur = dp[j];
                        l = q[j];
                    }else if(dp[j] == cur){
                        l += q[j];
                    }
                    //cur = Math.max(cur, dp[j]);
                   
                }
            }
            dp[i] = cur + 1;
            q[i] = l;
            if(dp[i] > maxVal){
                maxVal = dp[i];
                ans = q[i];
            }else if(dp[i] == maxVal){
                ans += q[i];
            }
        }
        return ans;
    }
}

```
