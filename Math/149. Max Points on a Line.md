## Problem description
```
Given n points on a 2D plane, find the maximum number of points that lie on the same straight line.

Example 1:

Input: [[1,1],[2,2],[3,3]]
Output: 3
Explanation:
^
|
|        o
|     o
|  o  
+------------->
0  1  2  3  4
Example 2:

Input: [[1,1],[3,2],[5,3],[4,1],[2,3],[1,4]]
Output: 4
Explanation:
^
|
|  o
|     o        o
|        o
|  o        o
+------------------->
0  1  2  3  4  5  6
```

# GCD
```java
class Solution {
    public int maxPoints(int[][] points) {
        // corner case
        int n = points.length;
        if (n <= 2) {
            return n;
        }
        int ans = 0;
        //iterate 1 point to all the other points
        for (int i = 0; i < n; ++i) {
            int[] cur = points[i];
            int dup = 1; 
            HashMap<Pair<Integer, Integer>, Integer> count = new HashMap<>();
            int maxCur = 0;
            for (int j = i + 1; j < n; ++j) {
                int[] sec = points[j];
                if (cur[0] == sec[0] && cur[1] == sec[1]) {
                    ++dup;
                } else {
                    Pair<Integer, Integer> s = slope(cur, sec);
                    count.put(s, count.getOrDefault(s, 0) + 1);
                    maxCur = Math.max(maxCur, count.get(s));
                }
            }
            ans = Math.max(ans, maxCur + dup);
        }
        return ans;
    }
    public Pair<Integer,Integer> slope(int[] a, int[] b) {
        // vertical line
        if (a[0] == b[0]) {
            return new Pair(0,a[0]);
        // horizontal line
        } else if (a[1] == b[1]) {
            return new Pair(a[1], 0);
        }
        int dx = a[0] - b[0];
        int dy = a[1] - b[1];
        int g = gcd(dx, dy);
        return new Pair(dx / g, dy / g);
    }
    public int gcd(int a, int b) {
        return b == 0 ? a : gcd(b, a % b);
    }
}

```
