## Problem description
```
Given a non-negative integer num represented as a string, remove k digits from the number so that the new number is the smallest possible.

Note:
The length of num is less than 10002 and will be â‰¥ k.
The given num does not contain any leading zero.
Example 1:

Input: num = "1432219", k = 3
Output: "1219"
Explanation: Remove the three digits 4, 3, and 2 to form the new number 1219 which is the smallest.
Example 2:

Input: num = "10200", k = 1
Output: "200"
Explanation: Remove the leading 1 and the number is 200. Note that the output must not contain leading zeroes.
Example 3:

Input: num = "10", k = 2
Output: "0"
Explanation: Remove all the digits from the number and it is left with nothing which is 0.
```

class Solution {
    public String removeKdigits(String num, int k) {
        if (k >= num.length()){
            return new String("0");
        }
        Stack<Integer> s = new Stack<>();
        //s.add(0);
        int i = 0;
        int n = 0;
        while(i < num.length() && n < k){
            int cur = num.charAt(i) - '0';
            //the first string
            if (s.size() == 0){
                s.add(cur);
                i++;
            //if there is one number and the following by 1
            } else if (cur == 0 && s.size() == 1){
                int last = s.pop();
                s.add(last * 10);
                i++;
            //if the number is larger than the top at the stack, we keep add number
            } else if (cur >= s.peek()){
                s.add(cur);
                i++;
            } else {
                s.pop();
                n++;
            }
            
        }
        while (n < k && s.size() > 0){
            s.pop();
            n++;
        }
        StringBuilder last = new StringBuilder(num.substring(i));
        StringBuilder front = new StringBuilder();
        while (s.size() > 0){
            int c = s.pop();
            Character b = (char)(c +'0');
            front.append(b);
        }
        front.reverse();
        front.append(last);
        return front.toString().isEmpty() ? "0" : front.toString();
    }
}
