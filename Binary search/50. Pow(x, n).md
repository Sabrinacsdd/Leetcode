## Problem description
```
Implement pow(x, n), which calculates x raised to the power n (xn).

Example 1:

Input: 2.00000, 10
Output: 1024.00000
Example 2:

Input: 2.10000, 3
Output: 9.26100
Example 3:

Input: 2.00000, -2
Output: 0.25000
Explanation: 2-2 = 1/22 = 1/4 = 0.25
Note:

-100.0 < x < 100.0
n is a 32-bit signed integer, within the range [−231, 231 − 1]
```

Approach 2: Fast Power Algorithm Recursive
```java
class Solution {
    public double myPow(double x, int n) {
        long N = n;
        if(N < 0){
            x = 1 / x;
            N = -N;
        }
        return pow(x, N);
    }
    private double pow(double x, long n){
        if(n == 0) return 1;
        double half = pow(x, n / 2);
        if( n % 2 == 0){
            return half * half;
        } 
        else{
            return half * half * x;
        }
    }
}

```

//Approach 3: Fast Power Algorithm Iterative
```Java
class Solution {
    public double myPow(double x, int n) {
        long N = n;
        if(N < 0){
            x = 1 / x;
            N = -N;
        }
        
        double ans = 1;
        double cur = x;
        for(long i = N; i > 0; i /= 2){
            if((i % 2 == 1)){
                ans = ans * cur;
            }
            cur = cur * cur;
        }     
        return ans;
    }  
}
```
